# swagsh

**A sleek, high-performance POSIX-compatible shell built in Rust for speed and reliability. Name inspired by 'swag' slang for stylish flair.**

## Installation

`swagsh` is built using Cargo.

```bash
git clone https://github.com/takashialpha/swagsh.git
cd swagsh
cargo build --release
````

The compiled binary will be available at:

```
target/release/swagsh
```

You may optionally install it system-wide:

```bash
sudo cp target/release/swagsh /usr/local/bin/swagsh
```

> ⚠️ Warning: replacing `/bin/sh` can be dangerous if you dont know what you're doing, make sure to test the shell before using it as your sh.
> Bugs can be found as the project is in constant development.
